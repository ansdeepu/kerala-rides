# **App Name**: Kerala Rides

## Core Features:

- Bus Search: Enable users to search for KSRTC buses by route or bus number.
- Real-Time Location Tracking: Display real-time locations of buses on an interactive map.
- Departure/Arrival Notifications: Allow users to set notifications for bus departures/arrivals at selected stops via WhatsApp.
- Share Bus Status: Enable users to share the current status of a bus with others.

## Style Guidelines:

- Primary color: Saturated red (#D32F2F) to represent KSRTC branding and ensure important elements stand out.
- Background color: Light gray (#F5F5F5) to provide a neutral and clean backdrop, ensuring readability and focus on the map and bus information.
- Accent color: Dark red (#B71C1C), a deeper shade of red, is used for interactive elements like buttons and active states.
- Headline font: 'Poppins' (sans-serif) for headlines, as this font provides a clean and contemporary appearance.
- Body font: 'PT Sans' (sans-serif) for body text to ensure readability.
- Use clear, vector-based icons for bus routes, stops, and notification settings to improve usability.
- Maintain a clean and intuitive layout, focusing on easy navigation and quick access to bus information.
